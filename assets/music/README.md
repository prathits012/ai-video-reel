# Background Music

Place royalty-free tracks here for use with `-m auto`:

| Filename   | Mood         |
|------------|--------------|
| calm.mp3   | Calm, ambient |
| uplifting.mp3 | Uplifting, inspirational |
| motivational.mp3 | Motivational, inspiring |
| meditation.mp3 | Meditation, peaceful |
| neutral.mp3 | Neutral, background |

Pixabay API does **not** support audio (only images/videos). Download manually:

1. Go to [Pexels Music](https://www.pexels.com/music/) or [Pixabay Music](https://pixabay.com/music/)
2. Search for each mood, download, and rename to match the filenames above

```bash
python scripts/download_music.py  # prints instructions
```
